# 🏡 Pakistan Real Estate AI Analyzer (End-to-End ML Project)

## 📌 Project Overview
An advanced AI-powered tool designed to analyze the Pakistan Real Estate market. This project integrates **Web Scraping** for real-time data collection and **Machine Learning** to provide accurate property price predictions.

## 🛠️ Tech Stack & Techniques

### 1. Web Scraping (Data Collection)
Utilized Python-based scraping scripts to extract market trends and property listings, ensuring the model stays updated with current market valuations.

### 2. Machine Learning (Prediction Engine)
Implemented the **Random Forest Regressor** algorithm. The model analyzes parameters like Area (Sq. Ft), Location, and City to predict prices with high precision.

### 3. Streamlit (Interactive UI)
Developed a user-friendly dashboard that allows anyone to predict prices and visualize trends through intuitive sliders and dropdowns.

### 4. Data Analytics
Leveraged **Pandas** and **NumPy** for robust data cleaning, preprocessing, and statistical analysis.

## 🚀 Key Features
* **Live Price Prediction:** Instant price estimation based on user input.
* **Multi-City Analysis:** Comprehensive data covering Karachi, Lahore, Islamabad, and more.
* **Interactive Visualization:** Dynamic charts and graphs for market trend analysis.
* **Efficient Research:** Reduces market research time for investors and buyers.

## 💻 How to Run Locally
1. Clone this repository to your local machine.
2. Install dependencies:  
   `pip install -r requirements.txt`
3. Launch the application:  
   `streamlit run Real_estate/main.py`
